# DigiTools – Premium Digital Tools Store

A full‑stack React application that matches the exact design from provided mockups. Includes product catalog, shopping cart, and responsive layout.

## Features

- **Pixel‑perfect design**: Navbar, banner, stats, product grid, steps, pricing, CTA, footer.
- **8 premium products** with icons, descriptions, features, and "Buy Now".
- **Cart functionality**: Add/remove items, total price, checkout with toast notifications
- **Fully responsive** (Tailwind CSS
- **Smooth scrolling**  + DaisyUI).

## Technologies

- React.js
- Tailwind CSS + DaisyUI
- React-Toastify
- JavaScript (ES6)
- JSON data

## Setup

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
